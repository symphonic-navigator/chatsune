# Grok 4.3 (with Grok 4.20 voice fallback) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Grok 4.20 and Grok 4.3 to the xAI adapter (with 4.3 falling back to 4.20-non-reasoning when thinking is off), expose an optional `remarks` field on `ModelMetaDto` to disclose the fallback in the model browser, and tighten Grok 4.1 Fast's advertised context window to 128 k.

**Architecture:** Replace the xAI adapter's two hard-coded slug constants with a per-`model_id` mapping that carries a `reasoning_slug`, a `non_reasoning_slug`, and an optional `remarks` string. `_build_chat_payload` resolves the outgoing API slug from this mapping using `request.model`; an unknown `request.model` falls back to the Grok 4.1 Fast entry (with a warning log) so legacy persona references stay routable. The frontend renders a third dimmed line under each model row when `remarks` is non-empty; the field defaults to `None` everywhere else (no migration needed).

**Tech Stack:** Python 3.12, FastAPI, Pydantic v2 (`shared/dtos/`), pytest with `pytest-asyncio` (backend tests), TypeScript + React (frontend), Vite + `tsc -b` (frontend build), `uv` for backend, `pnpm` for frontend.

**Spec:** `devdocs/specs/2026-05-01-grok-4.3-with-4.20-fallback-design.md`

```python
def _build_chat_payload(request: CompletionRequest) -> dict:
    entry = _XAI_MODELS_BY_ID.get(request.model)
    if entry is None:
        # Stale persona reference (e.g. legacy `xai:grok-4`) — fall back
        # to Grok 4.1 Fast so the request stays routable. Logged as a
        # warning so it shows up in Claude-oriented logs without
        # raising. The fallback path is exercised by Task 5's tests.
        _log.warning(
            "xAI: unknown model_id=%r in CompletionRequest; "
            "falling back to grok-4.1-fast",
            request.model,
        )
        entry = _XAI_MODELS_BY_ID["grok-4.1-fast"]

    model_slug = (
        entry.reasoning_slug if request.reasoning_enabled
        else entry.non_reasoning_slug
    )
    payload: dict = {
        "model": model_slug,
        "stream": True,
        "stream_options": {"include_usage": True},
        "messages": [_translate_message(m) for m in request.messages],
    }
    if request.temperature is not None:
        payload["temperature"] = request.temperature
    if request.tools:
        payload["tools"] = [
            {
                "type": "function",
                "function": {
                    "name": t.name,
                    "description": t.description,
                    "parameters": t.parameters,
                },
            }
            for t in request.tools
        ]
    return payload
```
